The African Network (TAN) Portal
Empowering the next generation of African scholars and tech-enabled professionals.
The African Network is a premium digital ecosystem and consultancy. We bridge the gap between high-potential African talent and global opportunities through a two-pronged approach: Global Mobility Strategy and High-Income Technical Upskilling.
